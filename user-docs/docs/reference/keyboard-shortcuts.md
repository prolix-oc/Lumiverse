# Keyboard Shortcuts

Quick reference for keyboard shortcuts in Lumiverse.

---

## Global

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + K** | Open Command Palette |
| **Escape** | Close current modal or panel |

---

## Chat

| Shortcut | Action |
|----------|--------|
| **Enter** | Send message |
| **Shift + Enter** | New line (without sending) |

---

## Command Palette

| Shortcut | Action |
|----------|--------|
| **Type to search** | Filter commands |
| **Arrow keys** | Navigate results |
| **Enter** | Execute selected command |
| **Escape** | Close palette |

---

## Navigation

The Command Palette (Cmd/Ctrl + K) provides keyboard-driven access to:

- Switch between panels
- Navigate to settings
- Open character browser
- Toggle features
- Run chat-specific actions
- Access extension tabs

---

## Tips

!!! tip "Command Palette is your friend"
    Almost everything in Lumiverse can be reached through the Command Palette. Learn to use it and you'll navigate the app much faster than clicking through menus.
